# Changelog

All notable changes to Query In will be documented here.

The project follows an issue-first workflow. Release entries should link the pull request or issue that introduced the change.

## Unreleased

- Added Podman-based container build files and GitHub Actions CI.
- Added Lighthouse CI with a practical bootstrap performance gate.
